# comitte

이 저장소는 국회 상임위원회 웹사이트에서 각 예산소위원회 명단을 수집해 하나의 표로 묶어 주는 간단한 파이썬 스크립트를 제공합니다. Google Apps Script로 작성하던 크롤러의 동작(쿠키 워밍업, EUC-KR 대응 등)을 그대로 옮겨 왔습니다.

## 준비 사항

- Python 3.10 이상
- `requests` 라이브러리 (`pip install requests`)

## 사용 방법

1. HTML 생성 스크립트를 실행합니다.

   ```bash
   python scripts/export_budget_table.py --output 예산소위.html
   ```

   네트워크 요청이 어려운 환경에서는 `--no-fetch` 옵션으로 빈 테이블을 생성하여 출력 형태만 확인할 수 있습니다.

2. 실행이 끝나면 지정한 경로에 `국회 예산소위원회 명단`이라는 제목의 HTML 파일이 생성됩니다. 노션이나 다른 웹페이지에 그대로 임베드하여 활용할 수 있습니다.

스크립트에 포함된 로직은 다음을 수행합니다.

- 각 상임위원회 URL을 순회하며 세션 쿠키를 선 요청으로 확보합니다.
- 포트 444가 닫혀 있을 경우를 대비해 동일한 URL에서 포트 번호를 제거한 주소도 재시도합니다.
- 응답 본문이 UTF-8로 깨져 보이면 EUC-KR로 다시 디코딩합니다.
- `subNmTb` 표에서 예산 소위원회 이름을 찾아내고, `polyCnTb` 표에서 정당별 의원 명단을 추출한 뒤 한 행으로 합칩니다.

이 모듈은 다른 파이썬 코드에서도 `comitte.scraper.fetch_budget_subcommittee_members()` 함수를 호출하여 바로 재사용할 수 있습니다.
